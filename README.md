# Portafolio Personal 

Portafolio profesional desarrollado con React para la materia de Manejo de Frameworks.

---

## Tecnologías

- React
- React Router DOM
- React Chart.js
- React Icons
- CSS3

---

## Instalación y Ejecución

```bash
# Paso 1: Crear el proyecto
npx create-react-app portafolio-app

# Paso 2: Entrar a la carpeta
cd portafolio-app

# Paso 3: Reemplazar la carpeta src
# Descarga la carpeta src de este repositorio y reemplaza la existente

# Paso 4: Instalar dependencias
npm install react-router-dom react-chartjs-2 chart.js react-icons

# Paso 5: Ejecutar el proyecto
npm start
```
